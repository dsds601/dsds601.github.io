# HTML (Hyper Markup Language)

* HTML이란?

  * 웹브라우저에서 문자열 이미지 동영상 입력양식 등을 출력하기 위해 사용 되는 언어
  * HTML은 실무적으로 **웹브라우저**가 **웹서버**에 접속한 후 **웹서버**가 **웹브라우저** 쪽으로 던져주는 언어
  * HTML안에는 DB 연동 결과물이 내재되어 있다.
  * HTML은 프로그램언어는 아니다 로직같은 형식이 없다.
  * **주의>**  개발자는 [입력양식] 태그를 조심해야한다. 오라클 / spring 하고도 연결되어 있다.

* HTML 차원에서 태그 란?

  * HTML안에서는 문자열,이미지 ,동영상 ,입력 양식등을 출력하기 위해 사용되는 형식을 말한다.

* 태그의 형식 

  * 태그의 형식1

    <tag명  속성명 ="속성값">
      컨텐츠 </tag명>

    * <태그명>과 </태그명>사이의 컨텐츠가 태그 기능의 영향을 받는다

    ```html
    EX)   <font color="blue">Bigdata</font> ->문자열이 파랑색으로 출력
    	  <b>Bigdata</b>   ->문자열이 두껍게 출력
    	  <center>Bigdate</center>  ->문자열이 웹브라우저 중앙위치로 출력
    ```

  * 태그의 형식2

  * <tag명 속성명 ="속성값"/>

  ```html
  EX) <hr>   ->수평선이 출력됨
  <input type="text" name="uid" size="10"> -> 문자열 입력하는 입력 양식
  ```

  * 태그명 --> 태그 종류명을 말한다. < />
  * **name** -> DB 컬럼명과 거의 일치한다. 서버쪽과도 연동이되어있어 중요하다.
  * 속성명 --> 태그의 성질을 말한다.  [type]
  * 속성값 --> 속성명에 대한 값을 말한다.   ["text"]
  * 속성명="속성값" 은 0개 이상 나올 수 있다.
  * 속성값은 " 또는 ' 로 감싼다. 만약 속성값 중간에 공백이 없으면 " 또는 '는 생략할 수 있으나 되도록 사용한다.
  * 컨텐츠에는 문자열 또는 또 다른 태그가 나올 수 있다.
  * 태그명, 속성명은 대문자 또는 소문자를 써도 된다. 되도록 소문자를 사용한다.

* table : 책장 만드는 태그

  * tr : 행을 만드는 태그
  * td : 행안에 셀을 만드는 태그 default 왼쪽정렬 설정
  * th : 행안의 셀 제목을 만드는 태그 default 가운데 정렬 볼드 설정

  

* input : 정보를 넣을 데이터나 정보를 보낼수있는 submit 만들 수 있는 태그

  * type : "text / emai / password " 등이 있다. -> db 에 담겨서 보내진다

  * type : "submit " 등으로 클릭하여 action에 url 열어 보내는 기능을 활 성화 시킬수있다,.

  * 서버에 보내기위해선 form action ="서버 url 주소" method="post / get "등 방식을 설정해야 한다. 

  * reset :  input type="reset"  **form 태그안에 있는 정보**를 초기화 시켜주는 기능

    * form태그 안에 reset이 없으면 초기화 되지 않습니다.

* HTML의 기본 구조 관련 태그는?

  ```html
  <html>
  <head>
  	<meta...>
  	<title> </title>
  </head>
  <body>...</body>
  </html>
  ```

  * html태그 : 소스 전체를 감싸는 태그

  * head태그 : HTML 문서의 제목이나 머리말을 감싸는 태그 , 자바스크립트 함수가 삽입되는 곳

  * title 태그 : 웹브라우저의 제목 표시줄에 나타나는 내용을 감싸는 태그

    * <head></head> 태그 안에 삽입된다.

  * body 태그 : 웹브라우저 화면에 출력될 컨텐츠(문자열 , 이미지 ,입력양식)를 감싸는 태그

  * meta 태그 : [화면이동] , [문자셋] , [검색 엔진이 검색하는 문자열] 등을 설정하는 태그이다.

* **웹서버** 구축후 웹서버에 내장된 파일끼리 , 즉 웹프로그램 자원끼리 호출할 때 자원경로 종류? ★
  * **절대경로**
  * "http://~:8080/a/b/xxx.jsp"  ->전세계 유일한 url 주소인 "http://~" 에 존재하는 웹서버 내에 a/b/xxx.jsp 경로에 있는 xxx.jsp 파일 호출하기  포트번호 앞까지의 주소를 도메인 주소라고 부른다. 포트번호 뒤의 주소는 웹서버내의 주소를 말한다. 포트번호는 대개 생략된다. 도메인 주소는 대부분 com 또는 net 또는 co.kr 또는 go.kr 등으로 끝난다.
  * "**/**a/b/xxx.jsp"   -> 동일한 웹서버 내에서 최상위 폴더 /a/b/xxx.jsp 경로에 있는 파일 호출하기  
    * **상대 경로**
    * "a/b/xxx.jsp"   -> 호출 파일이 속한 폴더내에 a 폴더내에 b폴더내에 xxx.jsp 파일 호출하기
    * ../xxx.jsp           -> 동일한 웹서버에 내에서 호출 파일이 속한 폴더를 벗어나서 존재하는 피호출 파일 xxx.jsp  호출하기
    * "../ㅇ/xxx.jsp"     -> 동일한 웹서버 내에서 호출 파일이 속한 폴더를 벗어나서 존재하는 d 폴더내에 피호출 파일 xxx.jsp 파일 호출하기
    * xxx.jsp -> 동일한 웹서버 내에서 호출파일이 속한 폴더 안에 존재하는 피호출 파일 xxx.jsp 파일 호출하기



* 웹 개발시 [웹디] [웹 퍼블리셔] [웹 프로그래머]의 역활

    * 포토샵 , 일러스트레이터를 사용하여 웹화면에 출력되는 이미지의 그래픽 작업을 한다.

    * HTML CSS Javascript 를 사용하여 웹화면의 UI(User Interface) 제작 작업을 한다.
    * **<참고>**  Javascript는 프로그래밍 언어의 한 종류이므로 [웹 퍼블리셔]는 프로그래머로서의 역량이 더 많이 필요하다고 볼수 있다.
  * 웹프로그래머
    * 서버쪽 언어 (JSP , JAVA ,PHP , ASP , .NET) 을 사용하여 UI로부터 또는 UI로의 데이터의 이동 , DB 연동 작업을 한다.
    * 입력양식 관련 HTML은 개발자가 할 수 도 있다.
    * 복잡하고 어려운 Javascript 작업을 하는 경우도 있다.



* 문단 정리 관련태그

  ```html
  <br>
  ```

  * 줄을 바꾼다 [엔터] 효과
  * **주의**  소스 상에서의 줄바꿈은 웹브라우저 출력 시 영향을 미치지 않는다.
  * 연이어 코딩한  br> 개수 만큼 줄바꿈이 발생한다.

  ```html
  <hr>
  ```

  * 한 행을 차지하는 수평선을 출력한다.

  ``` html
  <center> 문자열 </center>
  ```

  * 가운데로 정렬시킨다.

  ```html
  <pre> 문자열 </pre>
  ```

  * 사이의 공백이나 줄 바꿈을 그대로 인정해서 출력한다.

  ```html
  <p>
  ```

  * 한 행을 비운다.
  * **<주의>**  p 태그를 연이어 코딩할 경우 1개만 인정 된다.



* 글자 관련 태그는?

  ```html
  <h n개>
  </h n개>
  ```

  * n 자리에 1~6까지의 숫자를 넣어 문자열의 크기를 조절한다.
  * 문자열이 한행을 독차지하고 전후에 행도 비운다. 즉 3행을 독차지 한다.
  * 자동으로 문자열이 두꺼워진다.

  

  ```html
  <font></font>
  ```

  * 문자열의 글꼴 , 크기 , 색상등을 조절한다
  * font size="글자크기"  face="글꼴명"  color="글색상">에버랜드</font
    * -> 글자크기 1~7만 사용 숫자 클수록 글자가 크다
    * **<주의>**   생략시 size="3" 설정

  ```html
  <i>문자열</i>
  <em></em>
  <cite></cite>
  <var></var>
  ```

  * 문자열을 이탤릭체로 바꾼다

  ```html
  <code>문자열</code>
  <kbd></kbd>
  <tt></tt>
  ```

  * 문자열을 타자기체로 바꾼다.

  ```html
  <b>문자열</b>
  <strong></strong>
  ```

  * 문자열을 굵은 글씨로 바꾼다.

  ```html
  <sub>문자열</sub>
  ```

  * 문자열을 아래 첨자로 바꾼다.

  ```html
  <sup>문자열</sup>
  ```

  * 문자열을 위 첨자로 바꾼다.

  ```html
  	<u>문자열</u>
  ```

  * 문자열에 밑줄을 긋는다.

  ```html
  <s>문자열</s>
  <strike>문자열</strike>	
  ```

  * 문자열 가운데 선을 긋는다.

  ```html

  ```

* a 태그 사용 용도

  * [문자열]이나 [이미지 태그]을 감싸고 [클릭]등과 같은 이벤트를 발생하면 다음의 기능을 구현하는 태그이다.
    * [페이지 이동]
    * 같은 문서내 특정위치 이동
    * 파일 다운로드
    * 이미지 출력
    * 음악 동영상 실행
    * 자바스크립트 코드 실행
    * MS 아웃룩 실행 등
  * **<참고>**  기본적으로 문자열에 마우스를 대면 자동으로 손모양이 보인다.
  * **<참고>**  이벤트
    * 웹상에서 이벤트란 웹브라우저에서 발생하는 모든 움직임을 말한다.
    * 주로 마우스나 키보드에서 발생하지만 다른 것에 의해서도 발생한다.
      * 예> html 문서가 웹브라우저에 로딩되는것 html 문서가 웹브라우저에 언로딩 되는것(즉 사라지는것)
      * load , unload , click , dbclick , mouseover , mouseout , mousedown , mousedragged , keydown , keyup , keypress , focus , blur , **change** 등등

  ```
  <a href ="서버 자원의 url 주소">문자열</a>
  ```

  * 서버 자원의 URL 주소 호출로 새로운 페이지로 이동 , 즉 새 화면 열기

  ```html
  <a href ="javascript:자바스크립트코드">문자열</a>
  ```

  * 클릭시 코딩한 [자바스크립트코드] 실행된다.
  * 보통 [자바스크립트 코드] 는 함수 호출 코딩이다.

  ```html
  <a href="다운로드 파일명">문자열</a>
  ```

  * [파일 다운로드 대화상자]가 나타나고 확인 버튼 누르면 파일이 다운로드 된다.
  * **<주의>**  웹에서 열어볼 수 있는 파일은 [파일 다운로드 대화상자]가 뜨지 않고 새화면을 열고 실행한다.
    * **EX)** 그림파일 , txt파일 등

  ```html
  <a href="그림파일명">문자열</a>
  ```

  * 이미지만 있는 화면으로 이동

  ```html
  <a href="음악 동영상 파일">문자열</a>
  ```

  * 방문자에 설치된 프로그램으로 음악 동영상 파일 재생

  ```html
  <a href="#같은 화면내 이동 위치명">문자열</a>
  
  <a name ="xxx">상단</a>
  ```

* div 태그란?

  * 이미지 , 문자열 등을 기준별로 그룹화 시켜 서로 **[겹치게 포개 놓기] , [펼쳐 놓기] , [감춰 놓기]**등을 할 수 있는 태그

  * div>~</div  영역은 block 형식(한 행을 차지함 = 아래로 쌓인다.)   inline형식 ->왼쪽에서 오른쪽으로 쌓임

  * div> 태그의 속성은 css 표현방식인 style 속성으로 표현한다.

    ```html
    <div style="속성1 : 속성값1;~;">	
    ```

  * style 속성

    * position :static;                 -> div 태그의 현재 코딩 위치에서 그대로 출력됨deafult 설정 적지 않아도 됨
    * position:relative;              ->div 태그의 현재 코딩 위치를 0,0으로 기준 삼고 x축 y축으로 이동하여 다른 컨텐츠 위에 위치한다.
    * position:absolute             ->화면 왼쪽 상단 모서리를 0,0으로 기준 삼고 x축,y축으로 이동하여 다른 컨텐츠 위에 위치한다.
    * left : 화면 왼쪽 여백         ->웹브라우저 왼쪽 테두리에서 div 태그영역(div태그)까지의 가로 위치를 나타냄  **=>** 이게 먹히기위해선 position:absoulte 또는 position:relative 가 나와야한다.
    * top : 화면 위쪽 여백         ->웹브라우저 위쪽 테두리에서 div 태그영역(div태그)까지의 세로 위치를 나타냄  **=>** 이게 먹히기위해선 position:absoulte 또는 position:relative 가 나와야한다.

    * padding : 상여백 , 우여백 , 하여백 , 좌여백

      * => div 태그 영역 내부의 상 , 우 ,하 , 좌 여백 지정

      * => 상 , 우 ,하 ,좌 여백이 같은 값이면 padding : 여백; 으로만 써도 된다.

      * => 상 , 우 ,하 ,좌 여백을 따로 주고 싶으면 아래처럼 하면 된다.

        ```css
        padding-top:여백
        padding-right : 여백
        padding-bottom:여백
        padding-left:여백
        ```

    * width:div  태그 좌우 영역 넓이

    * height : div 태그 위아래 영역 넓이

    * border : 테두리 선두께 선모양 색상;

      * => div 태그 영역의 테두리선 두께 , 선모양 , 색상 설정 . 예>border : 2px solid red

    * background-color : div 태그 영역 색상;

    * z-index : 우선순위 번호

      * => div태그 영역이 겹쳤을 때 우선순위를 지정 , 번호가 클수록 상위.

    * visibility : 보임여부 설정

      * => div 태그 영역 보임여부를 설정, 보임 여부 설정값은 visible(보임) , hidden(안보임)
      * **<주의>**  hidden 설정 시 안보이고 차지하는 **공간은 남아 있음**

    * display : 보임여부 설정

      * => div 태그 영역 보임여부를 설정. 보임여부 설정값은 block(보임)  , none(안보임)
        * **<주의>**  none 설정시 안보이고 차지하는 **공간도 없음**

* table 태그 , td 태그 , tr 태그가 뭐 할 때 사용하나요

  * 문자열 , 또는 이미지를 [행]과 [열]로 표시하는 태그다. 즉, 컨텐츠의 배치를 관리하는 태그이다.

    * n행 m열의 행과 열을 만들어 컨텐츠를 배치하려면 아래 형식을 가진다.

    ```html
    <table>
    		<caption>표제목</caption>
    		<thead>
    			<tr><th>해더1</th><th>헤더2</th>~<th>헤더m</th></tr>
    		</thead>
    		<tbody>
    			<tr><td>데이터11</td><td>데이터12</td>~<td>데이터1m</td></tr>
    			<tr><td>데이터21</td><td>데이터22</td>~<td>데이터2m</td></tr>
    			<tr><td>데이터31</td><td>데이터32</td>~<td>데이터3m</td></tr>
    			.....
    			<td>데이터n1</td><td>데이터n2</td>~<td>데이터nm</td>
    		</tbody>
    	</table>
    
    ```

    * tr  의 개수가 전체 행의 개수이다.
    * 한 행의 <td  또는 th의 개수가 열의 개수이다.
    * caption 태그는 생략이 가능하다
    * thead tbody 태그도 생략 가능하다 그러나 JQuery 에서 테이블을 컨트롤 할 때는 유용하게 사용된다.
    * </td <th <tr 는 생략 가능하나 영역을 나타내므로 되도록 쓴다.
